# amairo-dev

## apps

- `portfolio`: ポートフォリオサイト
- `letter`: Web上で手紙を書いて画像として保存できるアプリ
- `poker`: プランニングポーカーアプリ

## tech stack

- [Turborepo](https://turborepo.org/) - モノレポ管理
- [Biome](https://biomejs.dev/) - コードフォーマッティング
- [Renovate](https://docs.renovatebot.com/) - 依存関係の自動更新

## development

```bash
pnpm turbo dev
```
